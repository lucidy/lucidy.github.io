MacOS中配置Apache及虚拟机

这篇博文思路大部分源自网络, 包括部分文字

MacOS自带Apache服务

什么是Apache?

1. 如何启动mac上的Apache?

sudo apachectl start # 启动
sudo apachectl restart # 重启
sudo apachectl stop # 停止
sudo apachectl -v # 查看Apache版本

2. 如何访问?

打开浏览器, 输入:http://localhost, 如果成功, 屏幕显示 It Works!

3. Apache的根路径在哪里?

/Library/WebServer/Documents/

4.在这里写自己的.php文件可以访问吗?

洗洗睡吧, 虽然可以, 但是权限问题和安全问题不好处理.

那么能不能配置一个自己的根目录, 然后外界访问的时候都来访问该目录呢?
虚拟机可以完成这个功能.
每个虚拟机都可以当做是一台独立的机器, 我们输入不同的域名就进入不同的目录.

Apache的安装目录在：/etc/apache2/，etc默认是隐藏的。

有三种方式查看：
终端下输入: cd /etc ; open .
dock下右键Finder，选择”前往文件夹”，输入”/etc”
可以在terminal 输入 “open /etc”

设置虚拟主机

1.在终端运行“sudo vi /etc/apache2/httpd.conf”，打开Apche的配置文件,
在httpd.conf中找到“#Include /private/etc/apache2/extra/httpd-vhosts.conf”，去掉前面的“＃”, 保存并退出.
2.运行“sudo apachectl restart”，重启Apache后就开启了虚拟主机配置功能.
